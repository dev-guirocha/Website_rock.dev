Projeto MedControl

Sistema de Gestão de Tratamento e Medicação.

CLIENTE: Sistema Único de Saúde (SUS)

TIPO: App Mobile & Sistema de Gestão

STACK: Swift, Node.js, SQL

O Desafio

O MedControl foi desenvolvido para lidar com um tipo de informação onde falhas não representam apenas inconsistências técnicas, mas riscos reais: acompanhamento de tratamentos de saúde. O objetivo do projeto não era criar apenas um aplicativo de lembretes, e sim estruturar um sistema confiável para organização contínua de rotinas médicas.

O desafio central envolvia representar corretamente o tempo. Medicamentos não são apenas registros, eles possuem frequência, intervalo, dependência de eventos e histórico acumulativo.

A Solução Rock .dev

A aplicação foi projetada para transformar essas regras em um fluxo verificável, garantindo que o estado atual do tratamento seja sempre coerente com o histórico registrado.

A arquitetura separa claramente interface e domínio, permitindo que a lógica de controle de doses e acompanhamento funcione independentemente da navegação do usuário. O sistema mantém histórico completo de tomadas, consultas e dados clínicos, possibilitando revisão posterior e compartilhamento estruturado das informações.

Além do uso individual, a aplicação foi preparada para múltiplos perfis, permitindo que um usuário gerencie tratamentos de terceiros sem comprometer a organização dos dados. O armazenamento persistente e as notificações garantem continuidade da rotina mesmo sem interação constante com o aplicativo.

Missão Cumprida: O Impacto

O resultado é uma ferramenta orientada à previsibilidade: o usuário não depende de memória para seguir um tratamento, mas de um sistema que representa corretamente o que deve acontecer ao longo do tempo.

Este projeto demonstra capacidade de modelar software em domínios sensíveis, onde consistência de estado, histórico confiável e comportamento previsível são mais importantes que a interface em si.

  • Controle temporal de doses com estado sempre coerente ao histórico
  • Separação entre domínio e interface para evolução segura da regra clínica
  • Histórico completo e notificações persistentes para continuidade do tratamento
