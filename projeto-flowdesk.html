Projeto FlowDesk

Sistema Operacional para Profissionais Autônomos.

CLIENTE: Produto Próprio / Profissionais Autônomos

TIPO: App Mobile de Gestão Operacional

STACK: React Native, Expo, AsyncStorage

O Desafio

O FlowDesk nasceu de um problema recorrente em negócios de serviço: a operação diária fragmentada. Agenda, clientes, cobranças e tarefas normalmente ficam distribuídos entre aplicativos diferentes, gerando retrabalho, atrasos e perda de controle financeiro.

O objetivo do projeto não foi criar mais um aplicativo de gestão, mas consolidar o fluxo de trabalho em um único ciclo operacional contínuo. Cada ação executada pelo usuário influencia diretamente a próxima etapa da rotina, eliminando a necessidade de reconciliação manual entre ferramentas.

A Solução Rock .dev

A arquitetura foi construída priorizando consistência de estado e rapidez de interação. Em vez de múltiplas telas independentes, o sistema opera como uma linha de execução: compromissos geram registros financeiros, registros financeiros atualizam previsões e a previsibilidade orienta decisões do dia seguinte.

Grande parte da complexidade do projeto está na modelagem das regras, não apenas armazenar dados, mas representar a rotina real do profissional. O aplicativo foi estruturado para funcionar com mínimo atrito cognitivo, permitindo uso rápido durante o trabalho sem exigir atenção exclusiva ao sistema.

Missão Cumprida: O Impacto

O resultado é uma ferramenta que deixa de ser apenas registro histórico e passa a atuar como apoio operacional. O usuário não consulta o aplicativo para ver o que aconteceu, ele consulta para saber o que precisa fazer agora.

Este projeto evidencia capacidade de transformar processos reais em software utilizável, unindo engenharia, UX funcional e pensamento de produto.

  • Fluxo operacional contínuo com agenda, clientes, cobranças e tarefas conectados
  • Consistência de estado entre compromissos, finanças e previsões diárias
  • Uso rápido no contexto de trabalho com baixo atrito cognitivo
