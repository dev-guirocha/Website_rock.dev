MedControl

Sistema de Gestão de Tratamento e Medicação.

CLIENTE: SUS / Saúde

TIPO: App Mobile + Gestão

STACK: Swift, Node.js, SQL

Ver código Falar comigo

O desafio

  • Representar frequência e intervalo de doses sem ambiguidade.
  • Manter histórico clínico confiável para revisão posterior.
  • Garantir continuidade da rotina sem depender de memória.

Em domínio sensível, inconsistência não é só bug: é risco real. O projeto exigiu modelagem temporal rigorosa para manter o estado do tratamento sempre coerente com o histórico registrado.
